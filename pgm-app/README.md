# PGM — Production Grand Master

Multi-camera live production switcher with real-time tally lights.

## Pages

- `/` — Home
- `/director` — Director console (open on Mac)
- `/camera?id=cam1&name=Cam+1&label=Stage+Left` — Camera page (open on iPhone)

## Development

```bash
npm install
npm run dev
```

## Deploy

Deployed on Vercel. Push to main branch to trigger a deploy.
